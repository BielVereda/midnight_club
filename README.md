# 🚗 Midnight Club 3 - Car Showcase

Projeto front-end inspirado no universo de corridas urbanas, apresentando alguns dos carros mais icônicos do jogo *Midnight Club 3*.

---

## 📸 Preview

Interface moderna com estilo neon, exibindo carros em cards responsivos com design limpo e visual inspirado em corridas de rua.

---

## 🚀 Tecnologias utilizadas

* ⚛️ React
* 🎨 CSS3 (com responsividade)
* 📦 Vite

---

## 🧠 Funcionalidades

* ✅ Exibição de carros em formato de cards
* ✅ Layout responsivo (Desktop, Tablet e Mobile)
* ✅ Estilo visual com tema escuro + neon
* ✅ Componentização com React
* ✅ Uso de imagens locais

---

## 🛠️ Como rodar o projeto

```bash 
# Clone o repositório
git clone https://github.com/BielVereda/midnight_club.git

# Acesse a pasta
cd midnight-club3

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
